비대면 캠 스터디 사이트 Catudy

Catudy란,

캠(Cam)과 스터디(Study)를 결합한 합성어로, 비대면 시대에 맞추어 제작하게 된 캠 스터디 사이트입니다.

📍 프로젝트 기간 : 2022/07/31 ~ 2022/08/30 (약 1개월)

💡 요약

두번째로 만들었던 팀 프로젝트입니다. 비록 완성은 하지 못 했지만 반응형 웹과 Spring Boot를 공부할 수 있었던 좋은 시간이었습니다. 저는 주로 UI를 담당하여 페이지를 디자인했으며, 캠 스터디라는 특성상 사용자들이 휴대폰과 노트북, 컴퓨터 등 다양한 방법으로 접속하는 것을 고려하여 반응형 웹을 구현해보려고 노력했습니다.

💡 주요 기능

  • 메인 페이지(UI) - 사용자가 속한 스터디 정보, 현재 진행 중인 캠 스터디 목록, 현재 모집 중인 스터디 목록
  • 모집 페이지(UI) - 4개의 카테고리(모집중, 모집 완료, 질문&답변, 자유)로 나뉘어진 스터디 모집 게시판
  • 유저 회원가입(정규 표현식을 이용한 유효성 검사, Ajax) 및 로그인 기능

💡 사용 언어 및 기술

  • Back-end : Java 11
  • Front-end : HTML, CSS, JavaScript(Jquery), Thymeleaf
  • DBMS : MySQL(Mybatis)
  • Spring Boot

💡 사용 프로그램

  • Eclipse
  • Visual Studio Code
  • Github
  • DBeaver
